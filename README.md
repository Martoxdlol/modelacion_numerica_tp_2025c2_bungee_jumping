# Trabajo Práctico - Bungee Jumping

En este repositorio se incluye la solución al trabajo práctico de Modelación Numérica 2025C2 "Análisis cinemático y dinámico de un salto Bungee Jumping", que consiste en modelar y simular el salto bungee jumping utilizando diferentes métodos numéricos.

## Contenidos del Repositorio

- `informe/`: Informe (en .md o .pdf) detallado del trabajo práctico, incluyendo la formulación del problema, los métodos numéricos utilizados, los resultados obtenidos y las conclusiones.
- `src/`: Código fuente utilizado para las simulaciones y análisis.
- `outputs/`: Carpeta que contiene las imágenes y gráficos generados durante el análisis y simulaciones.

## Instrucciones para Ejecutar el Código

1. Instalar dependencias necesarias:
   1. Python 3.x
   2. Librerías: NumPy, Matplotlib (pueden instalarse usando pip)
   ```bash
   pip install numpy matplotlib
   ```

2. Navegar al directorio `src/`:
   ```bash
   cd src/
   ```
3. Ejecutar los scripts de los diferentes puntos:
   ```bash
   python3 punto_3.py
   python3 punto_4.py
   python3 punto_5.py
   python3 punto_5y7.py
   python3 grafico_punto_6.py
   python3 grafico_punto_7.py
   python3 simulacion_punto_6y7.py
   ```
